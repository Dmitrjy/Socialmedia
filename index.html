socialmedia.js

This page demonstrates the use of socialmedia.js file template. Using this JavaScript template, you can add facebook and twitter core functions to your application without messing a lot of code.

Include this file using following code into your application.

<script src="socialmedia.js"></script>

Add following script and provide the app_id.

<script>
        // replace app_id with your application id e.g. '1234567890'
        facebook.init( app_id ); 
</script>

There are two predefined objects facebook and twitter that can be used to call different relevant methods.

Example use of three available methods are:

  • facebook.share( name, link, picture, caption, description );
  • facebook.invite( message );
  • twitter.tweet( message, handler, url );

Examples

Share to Facebook timeline 

<script>
$('sharebutton').on('click', function()	{
	
    facebook.share( name, link, picture, caption, description );

});
</script>

Invite friend on Facebook 

<script>
$('invitebutton').on('click', function() {

	facebook.invite( message );            

});
</script>

Tweet a story 

<script>
$('tweetbutton').on('click', function()	{

	twitter.tweet( message, handler, url );            

});
</script>
